# SandSimulation 

#### Building
There are a few libraries which are required to build this project. (I've included them on the vendor folder for convenience)
 - GLFW
 - GLEW
 - GLM

#### Controls
 - C: Clear Screen
 - Left Click: Place Material
 - Right Click: Erase Material
 - 1: Material Sand
 - 2: Material Water
 - 3: Material Wood
 - 4: Material Fire
 - 5: Material Smoke
 - 6: Material Oil
 - 7: Material Acid
